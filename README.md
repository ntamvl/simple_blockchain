# Simple BlockChain
Simple BlockChain implement by Go Lang

## How to build
```
cd && git clone https://github.com/ntamvl/simple_blockchain.git
cd simple_blockchain 
go get
go build
```

## How to run 
```
./simple_blockchain
```
